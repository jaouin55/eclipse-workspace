1)변수와 자료형 : 개발자가 컴퓨터에게 일거리를 제공하는 법 제공방법 : 메모리에 올려야 함 cf.사람은 기억력을 사용 [자료형] 게발자는 컴퓨터가 이해할 수 있는 데이터 형태로 제공 해야 하는데, 프로그래밍 언어에서 사용할 수 있는 자료형은 3가지 밖에 없다..(숫자,문자,논리값) [변수] :전산분야에서는 메모리에 생성된 공간 (수학적으로는 변하는 수를 의미) 변수값은 언제나 개발자에 의해 변할 수 있다. 개발자가 데이터를 메모리에 올리는 법: 변수 초기화 ex) var a = 5; // 변수 a를 선언하면서 그 값을 5로 할당 즉, 5로 초기화하였다 라는 뜻 [변수명] :메모리에 생성된 공간을 접근하기 위해 붙인 이름 선언시 금지 규칙 (모든 언어 공통) 1) 숫자로 시작할 수 없다. 2)특수 문자를 사용할 수 없다. cf>허용되는 2가지 ($(달러) , _(언더바)) 3)한글을 사용하지 말 것. 2)연산자 :메모리에 올려진 일거리를 어떻게 처리할지 결정하는 부호 산술연산자 : + - / * 3)제어문 :조건 / 반복을 처리하여 프로그램을 효율적으로 만들기 4)함수 :내가 이미 작성한 코드를 재사용 하기 위해 사용 5)배열 :대량의 데이터를 대상으로 효율적인 업무 가능 6)객체 :프로그램으로 현실세계의 사물,개념을 표현가능 ex)총알 , 적군 , 주인공 , 아이템 등.. 7)객체의 활용 ---------------------------------------------------- 자료형 변수명 = 할당데이터 var(variable) - 프로그램이 변수 자료형을 알아서 판단..
